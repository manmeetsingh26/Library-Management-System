# 📚 Library Management System

A C++ based Library Management System developed to manage books, student records, and borrowing operations through a menu-driven interface.

This project is part of my software development learning journey and focuses on applying programming fundamentals to a practical real-world system.

---

## 📖 About The Project

The Library Management System is designed to manage basic library operations such as adding books, viewing book records, issuing books, returning books, and maintaining borrowing details.

This project helps in understanding:

- Record Management
- Data Storage
- Searching Techniques
- Menu-Driven Programming
- File Handling
- Software Development Workflow

---

## 🚀 Planned Features

### ✅ Basic Menu System
A simple console-based menu for navigating different library operations.

### 🔄 Add Book
Add book details such as title, author, and book ID.

### 🔄 View Books
Display all available books in the library.

### 🔄 Search Book
Search books using book ID or book title.

### 🔄 Issue Book
Issue a book to a student and store borrowing details.

### 🔄 Return Book
Return an issued book and update availability status.

### 🔄 File Handling
Save and load book records permanently.
